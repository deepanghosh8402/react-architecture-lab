# React Architecture Lab

## Overview
This repository is a structured learning and experimentation space for advanced React concepts and enterprise-grade frontend architecture.

The goal is to strengthen practical knowledge in modern React development, state management, project structuring, and Git workflows.

---

## Tech Stack
- React (JSX)
- Vite
- Redux Toolkit (Planned)
- React Router
- Modern ES6+
- Git Workflow using Sourcetree

---

## Objectives
- Implement advanced state management patterns
- Practice scalable folder architecture
- Understand debugging and build workflows
- Follow professional Git branching strategy
- Simulate production-style application structure

---

## Getting Started

### Install dependencies
npm install

### Run development server
npm run dev

### Build for production
npm run build
